# Master Case Studies

A modern, responsive web platform for mastering business case studies. Built with HTML5, CSS3, and vanilla JavaScript, it features:

- Structured case practice modules
- Clean, accessible design
- Mobile-first navigation and layout
- Downloadable resources and frameworks
- Modern branding and color system

## Live Demo
